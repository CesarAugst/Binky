1. Essa opção é indicada para usuários que desejam obter informações sobre o espaço
consumido na máquina que está usando no momento
        para utiliza-la basta teclar -1- e as informações serão exibidas. Tecle ENTER
        para retornar

2. Essa opção é indicada para usuários que desejam obter informações sobre processos
da máquina que está usando no momento
        para utiliza-la basta teclar -2- e as informações serão exibidas. Tecle ENTER
        para retornar

3. Essa opção é indicada para usuários que desejam obter informações sobre o uso do 
do processador da máquina que está usando no momento
        para utiliza-la basta teclar -3- e as informações serão exibidas. Tecle ENTER
        para retornar

4. Essa opção é indicada para usuários que desejam obter informações sobre o linux
da máquina que está usando no momento
        para utiliza-la basta teclar -4- e as informações serão exibidas. Tecle ENTER
        para retornar

5. Essa opção é indicada para usuários que desejam obter informações sobre o kernel
do linux da máquina que está usando no momento
        para utiliza-la basta teclar -5- e as informações serão exibidas. Tecle ENTER
        para retornar
